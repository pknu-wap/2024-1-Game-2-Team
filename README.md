# 덱 빌딩 어드벤처 로그라이크
<b>\<서울 2033></b>의 <b>스토리 진행</b>과 <b>\<Slay the Spire></b>의 <b>덱 빌딩</b>에 강하게 영감을 받은 게임입니다.

좀비 사태 발발 직후, 주인공은 딸이 납치되었단 사실을 알게 됩니다.

들끓는 좀비와 멸망한 세계의 사람에게서 생존하며, 다양한 일을 겪고, 최후의 전투를 위해 강해집니다.

과연 여러분은 좀비들과 싸우며, 무사히 딸을 구해낼 수 있을까요?
